# my simple koa/graphql server 👨‍💻💨

![demo-gif](demo.gif)

A super simple nodejs/koa server with a single graphql endpoint.
Currently using json-server (see db.json) for storing dummy data.

### Installing

First

```
yarn
```

then to run

```
yarn start
```
